# 9th Form Website

A modern, professional website for 9th Form - a growth strategy and brand building company with a streetwear skate aesthetic.

## Features

- **Modern React TypeScript Setup** - Built with React 18 and TypeScript for type safety
- **Professional Streetwear Design** - Clean, modern aesthetic with skate culture influences
- **Smooth Animations** - Framer Motion animations for engaging user experience
- **Fully Responsive** - Mobile-first design that works on all devices
- **Interactive Components** - Hover effects, smooth scrolling, and dynamic interactions
- **Professional Styling** - Styled Components for maintainable CSS-in-JS

## Tech Stack

- React 18
- TypeScript
- Styled Components
- Framer Motion
- Font Awesome Icons
- Google Fonts (Inter & Space Grotesk)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Design Features

- **Color Scheme**: Black, white, and cyan (#00d4ff) with orange accents (#ff6b35)
- **Typography**: Inter for body text, Space Grotesk for headings
- **Animations**: Smooth scroll-triggered animations using Framer Motion
- **Responsive**: Mobile-first design with breakpoints at 768px and 480px
- **Interactive Elements**: Hover effects, form validation, and smooth transitions

## Customization

The website uses your existing logo and gallery images. Make sure to place your images in the `public/images/` directory:

- `9thform_logo_white.png` - Main logo
- `9thform_logo_white_flame.png` - Hero logo
- `gallery/` - Portfolio images
